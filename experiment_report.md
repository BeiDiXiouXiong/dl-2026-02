# CIFAR-10 图像分类对比实验报告
## 一、实验目的
通过设计对比实验，分析不同超参数对模型训练效果的影响，诊断模型错误原因，并提出合理的改进方向，满足作业“训练结果诊断与改进实验”要求。

## 二、实验配置
本次实验基于 SimpleCNN 模型，以 CIFAR-10 为数据集，设计 2 组对比实验，核心变量为「学习率」和「是否使用数据增强」，其他参数保持一致。

### 实验基础配置（所有组共用）
| 配置项 | 具体值 |
| :--- | :--- |
| 模型 | SimpleCNN（Conv2d + MaxPool2d + Linear） |
| 数据集 | CIFAR-10（训练集50000张，测试集10000张） |
|  batch size | 64 |
| 优化器 | Adam |
| 损失函数 | CrossEntropyLoss |
| 训练轮次 | 5 epoch |
| 设备 | GPU（CUDA） |
| 数据预处理 | ToTensor() + Normalize((0.5,0.5,0.5), (0.5,0.5,0.5)) |

### 对比实验配置表
| 实验组 | 学习率 | 是否使用数据增强 | 核心变量 |
| :--- | :--- | :--- | :--- |
| 对照组 | 0.001 | 否 | 无（基础配置） |
| 实验组1 | 0.01 | 否 | 学习率（放大10倍） |
| 实验组2 | 0.001 | 是 | 数据增强（随机水平翻转+随机裁剪） |

#### 数据增强具体配置（实验组2）
```python
transform = transforms.Compose([
    transforms.RandomCrop(32, padding=4),  # 随机裁剪
    transforms.RandomHorizontalFlip(),     # 随机水平翻转
    transforms.ToTensor(),
    transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5))
])
```

### 3. 错误分类样本分析（对照组）
共选取 5 张错误分类样本（保存于 outputs/wrong_samples.png），具体分析如下：

| 错例序号 | 真实类别 | 预测类别 | 错误原因分析 |
| :--- | :--- | :--- | :--- |
| 1 | cat（猫） | dog（狗） | 猫和狗的外形特征相似（毛发、体型），模型未能捕捉到面部细节（如耳朵形状、眼睛大小），导致混淆 |
| 2 | bird（鸟） | airplane（飞机） | 鸟的翅膀与飞机的机翼外形相似，且背景为天空时，模型误将鸟的轮廓识别为飞机 |
| 3 | deer（鹿） | horse（马） | 鹿和马均为四足动物，体型相近，模型未能区分鹿的角和马的耳朵差异 |
| 4 | frog（青蛙） | cat（猫） | 青蛙的深色皮肤与猫的深色毛发颜色相近，且样本分辨率较低，特征不清晰，导致误判 |
| 5 | automobile（汽车） | truck（卡车） | 小型汽车与卡车的轮廓相似，模型未能区分两者的车身长度和车轮数量差异 |

## 四、模型错误原因总结
1.  特征提取能力不足：SimpleCNN 结构简单，仅包含2个卷积层，无法捕捉图像的深层特征，对外形相似的类别（如猫/狗、鹿/马）区分能力弱。
2.  数据分辨率限制：CIFAR-10 图像分辨率仅为32x32，细节信息少，部分样本特征模糊，导致模型难以提取有效区分特征。
3.  过拟合倾向：对照组训练准确率（78.32%）高于测试准确率（69.57%），存在轻微过拟合，模型对训练集特征拟合较好，但泛化能力一般。
4.  超参数影响：学习率过大（实验组1，0.01）会导致模型训练不稳定，收敛速度慢，准确率下降；数据增强（实验组2）能提升模型泛化能力，减少过拟合。

## 五、下一步改进方向
1.  优化模型结构：增加卷积层数量（如添加conv3、conv4），或使用预训练模型（如ResNet18），提升深层特征提取能力，解决外形相似类别的区分问题。
2.  优化超参数：调整学习率为动态学习率（如使用学习率衰减），避免固定学习率导致的训练不稳定；尝试不同的batch size（如32、128），观察对训练效果的影响。
3.  增强数据预处理：增加更多数据增强方式（如随机旋转、亮度调整），扩大训练集多样性，进一步提升模型泛化能力，缓解过拟合。
4.  正则化优化：在模型中添加 Dropout 层（如在全连接层后添加 nn.Dropout(0.5)），或使用 L2 正则化（weight decay），减少过拟合。
5.  延长训练轮次：将训练轮次增加至10-15个epoch，观察模型是否能进一步收敛，提升测试准确率。

## 六、实验结论
1.  学习率对模型训练效果影响显著：合适的学习率（0.001）能使模型稳定收敛，学习率过大（0.01）会导致训练不稳定、准确率下降。
2.  数据增强能有效提升模型泛化能力：实验组2（使用数据增强）的测试准确率（72.19%）高于对照组（69.57%），且训练更稳定，能缓解过拟合。
3.  SimpleCNN 模型能完成基本的图像分类任务，但特征提取能力有限，通过优化模型结构、超参数和数据预处理，可进一步提升准确率。
