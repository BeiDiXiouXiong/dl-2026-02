# Experiment Report - CIFAR-10 Image Classification

## 1. 实验目的

本实验旨在基于 PyTorch 实现一个基础图像分类模型，完成从数据加载、模型训练到结果评估的完整流程，并通过对比实验分析不同超参数对模型性能的影响。

---

## 2. 数据集说明

本实验使用 CIFAR-10 数据集：

- 图像尺寸：32 × 32 × 3
- 类别数量：10
- 包含类别：airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck

---

## 3. 模型结构

本实验采用简单卷积神经网络（CNN）：

- Conv → ReLU → MaxPool
- Conv → ReLU → MaxPool
- 全连接层（FC）
- 输出 10 类

该模型结构简单，但能够有效提取图像特征。

---

## 4. 实验设置

| 参数 | 数值 |
|------|------|
| Epoch | 5 |
| Batch Size | 64 |
| Loss | CrossEntropyLoss |
| Optimizer | Adam / SGD |
| Learning Rate | 0.001 / 0.01 |

---

## 5. 实验结果

### 实验1：不同学习率

| Learning Rate | Test Accuracy |
|--------------|--------------|
| 0.001 | 0.63 |
| 0.01 | 0.58 |

**分析：**

- 较小学习率（0.001）收敛稳定，表现更好  
- 较大学习率（0.01）更新过快，导致性能下降  

---

### 实验2：不同优化器

| Optimizer | Test Accuracy |
|-----------|--------------|
| Adam | 0.63 |
| SGD | 0.59 |

**分析：**

- Adam 收敛速度更快  
- SGD 在本实验中表现略差  

---

## 6. 训练曲线分析

训练过程中：

- Loss 持续下降，说明模型在学习  
- Train Accuracy 与 Test Accuracy 同步上升  
- 未出现明显过拟合现象

## 7. 错误分类分析

以下是模型常见错误类型：

1. **猫 vs 狗**
   - 原因：外观相似，纹理特征接近  

2. **飞机 vs 鸟**
   - 原因：背景影响较大，模型误判  

3. **小目标类别**
   - 原因：分辨率较低（32×32），细节不足  

4. **遮挡情况**
   - 模型难以识别被部分遮挡的物体  

5. **复杂背景**
   - 模型容易受到背景干扰  

---

## 8. 模型不足与改进方向

### 当前问题：

- 模型结构较浅，表达能力有限  
- 对相似类别区分能力不足  
- 对背景敏感  

---

### 改进方向：

1. 使用更深网络（如 ResNet）
2. 引入数据增强（flip、crop）
3. 使用学习率调度器
4. 增加 dropout 防止过拟合
5. 使用更大规模数据训练

---

## 9. 结论

本实验成功实现了一个完整的图像分类训练流程，并通过对比实验分析了不同超参数对模型性能的影响。结果表明：

- 合理选择学习率和优化器对模型性能至关重要  
- 简单 CNN 已能达到约 60%+ 准确率  
- 模型仍有较大提升空间
